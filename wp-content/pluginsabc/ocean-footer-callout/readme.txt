== Description ==

Add some relevant/important information about your company or product in your footer.
This plugin requires the [OceanWP](https://oceanwp.org/) theme to be installed.

== Installation ==

1. Upload `ocean-footer-callout` to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins > Installed Plugins' menu in WordPress
3. Done!

== Frequently Asked Questions ==

= I installed the plugin but it does not work =

This plugin will only function with the [OceanWP](https://oceanwp.org/) theme.
